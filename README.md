# cocos2dJsProject
一个简单的jumpChess游戏
一开始写象棋卡住了就先写西洋跳棋，感觉跳棋会简单点，结果发现跳棋更难2333~ 象棋没有这么复杂的强制吃棋规则，也没有连跳（一次走多不），也不会出现无棋可走的情况

还有一些规则没有实现（比如跳棋规则中如果能吃棋是必须强制吃棋的，但是这里一次默认是一次走一步而且没有强制去吃棋的规则，而且西洋跳棋是可以连跳的）
实习自己写游戏练手的阶段快结束了，以后看心情完善吧
